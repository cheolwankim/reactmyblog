## 💡 Introducing Project
트위터를 나만의 형식으로 구현한 프로젝트입니다<br>
회원가입을 한다면 누구든지 글을 작성하고 실시간으로 바로 확인할 수 있습니다.<br>
회원가입은 구글 Github 또는 새로운 계정을 생성해서 진행할 수 있습니다.<br>
자신이 쓴 모든 글들은 수정과 삭제가 가능하고 이미지 또한 올릴 수 있습니다.<br>
